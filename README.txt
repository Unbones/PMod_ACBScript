How to Use:
Copy the SpongebobSquarepants folder from your Xbox and copy the contents in the ".\Patcher\Game" Directory
(It Should Look Like Game\b1 NOT Game\SpongebobSquarepants\b1)
Download the Practice Mod at "https://github.com/TetraxZ/BfBB-Practice-Mod"
After Downloading Place it in the "Patcher" Folder and rename it to "PMod.zip"
Download the AutoCB zip from the BfBB Speedrun Discord
Place the ips file for AutoCB in the "Patcher" Folder and rename it to "acb.ips"
Finally, run "Patcher.bat" and it will automatically install everything to the "PracticeMod" Folder
